Seletor Universal No CSS

O que é o Seletor Universal no CSS?

O seletor universal no CSS é representado pelo caractere *. Ele é utilizado para aplicar estilos a todos os elementos da página, sem exceção. Isso significa que qualquer regra definida usando o seletor universal será aplicada em todos os elementos HTML presentes no documento.

Uma das utilizações mais comuns do seletor universal é para realizar o reset de margens e espaçamentos padrões dos navegadores, garantindo assim que todos os elementos comecem com os mesmos valores.
