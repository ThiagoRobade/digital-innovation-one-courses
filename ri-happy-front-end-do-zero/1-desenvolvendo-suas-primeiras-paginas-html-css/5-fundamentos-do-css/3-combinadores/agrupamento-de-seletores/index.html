Combinando Seletores no CSS

No CSS, você pode combinar diferentes tipos de seletores para aplicar estilos de forma mais específica e poderosa. Isso permite estilizar elementos de acordo com múltiplas condições, como classes, tipos de elementos, atributos e pseudoclasses.

Abaixo estão alguns exemplos de combinações de seletores:

Exemplo 1: Classe com Atributo

O seletor .highlight[type="text"] aplica estilos aos campos de texto que possuem a classe highlight:


    .highlight[type="text"] {
        border: 2px solid #FF9800;
    }

Exemplo 2: Tag com Pseudoclasse

O seletor button:hover aplica estilos ao botão quando o mouse passa por cima dele:


    button:hover {
        background-color: #4CAF50;
        color: white;
    }

Exemplo 3: Duas Classes

O seletor .btn.large aplica estilos a elementos que possuem ambas as classes btn e large:


    .btn.large {
        padding: 15px 30px;
        font-size: 18px;
    }

Exemplo 4: Seletor de Descendência

O seletor div p aplica estilos a todos os parágrafos que são descendentes de uma <div>:


    div p {
        color: #555;
        font-style: italic;
    }

Este parágrafo está dentro de uma div.
