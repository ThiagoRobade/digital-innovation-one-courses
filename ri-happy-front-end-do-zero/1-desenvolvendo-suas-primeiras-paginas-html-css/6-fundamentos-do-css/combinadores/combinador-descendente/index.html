O que é o Combinador Descendente no CSS?

O combinador descendente no CSS é utilizado para selecionar elementos que são descendentes de outro elemento. Um descendente é qualquer elemento que está aninhado dentro de outro, não necessariamente seu filho direto. No CSS, o combinador descendente é representado por um espaço entre dois seletores.

Por exemplo, o seletor div p irá selecionar todos os elementos <p> que estão dentro de um elemento <div>, independentemente de quão aninhados estejam. A seguir, temos exemplos práticos.

Exemplo 1: Parágrafos dentro de uma Div

O seletor div p estiliza os parágrafos que estão dentro de um elemento <div>:


    div p {
        color: #4CAF50;
        font-style: italic;
    }

Este parágrafo está dentro de uma div.

Outro parágrafo dentro da mesma div.

Este parágrafo está fora da div e não será afetado.

Exemplo 2: Spans dentro de um Container

O seletor .container span estiliza os elementos <span> que são descendentes de um elemento com a classe container:


    .container span {
        background-color: #FFEB3B;
        padding: 2px 4px;
        border-radius: 4px;
    }

Texto com destaque em amarelo.

Exemplo 3: Links dentro de Listas

O seletor ul li a estiliza os links (<a>) que estão dentro de itens de lista (<li>) dentro de uma lista não ordenada (<ul>):


    ul li a {
        text-decoration: none;
        color: #2196F3;
    }
