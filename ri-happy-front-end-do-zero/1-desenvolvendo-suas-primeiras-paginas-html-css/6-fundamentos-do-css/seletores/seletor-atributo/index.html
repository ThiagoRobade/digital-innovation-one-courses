O que é o Seletor de Atributo no CSS?

O seletor de atributo no CSS é usado para aplicar estilos a elementos HTML que possuem um atributo específico ou um valor de atributo específico. Ele é definido utilizando colchetes ([]) em volta do nome do atributo ou do atributo com um valor. Por exemplo, o seletor [type="text"] irá aplicar estilos a todos os elementos <input> que possuem o atributo type com o valor "text".

O seletor de atributo é muito útil quando você quer estilizar elementos com base em atributos e seus valores, como campos de formulário com diferentes tipos, links com diferentes destinos, entre outros.

Exemplo de uso do seletor de atributo no CSS: 

        
        [type="text"] {
            border: 2px solid #4CAF50;
        }

        [type="submit"] {
            background-color: #4CAF50;
            color: white;
        }

No exemplo acima, estamos estilizando um campo de texto e um botão de envio de formulário com base no atributo type de cada um.
